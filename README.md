# 没有你的天气
基于MVVM框架，使用Android Jetpack组件开发的一款天气APP
## 下载地址
https://www.coolapk.com/apk/com.weather
## 主要功能
- 显示未来七天的雨天
- 全天温度变化
- 实时天气
- 桌面插件
